# PhishGuard - AI Phishing Detection

## Overview
PhishGuard is an AI-powered phishing detection system that identifies phishing URLs, emails, and SMS messages using Machine Learning.

## Features
- Detect phishing URLs
- Detect phishing emails
- Detect phishing SMS
- Machine Learning based prediction
- User-friendly web interface using Flask

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML/CSS
- Joblib
- Pandas
- NumPy
